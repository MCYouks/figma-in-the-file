# figma-in-the-file

This template should help get you started developing a darkmode with Vue 3.

## Project demo

See live demo here: [figma.luego.eu](https://figma.luego.eu/)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
